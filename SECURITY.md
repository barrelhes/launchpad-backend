# Security Policy

## Supported Versions

This project only supports the latest version available on the `main` branch.

| Version | Supported |
| ------ | --------- |
| Latest | ✅ |
| Older | ❌ |

## Reporting a Vulnerability

If you discover a security vulnerability, please **do not open a public issue**.

Instead:
- Open a **private security advisory** on GitHub  
  or
- Contact the maintainer via GitHub

You can expect an initial response within **48–72 hours**.

If the vulnerability is confirmed, a fix will be worked on and released as soon as possible.
